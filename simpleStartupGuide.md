# Guide

## General Instructions

- Clone the full repo
- chemical_visualizer for backend
- chemical_visualizer_frontend for frontend
- desktop_app for PyQt5 desktop client
- create seperate venv for each folder chemical_visualizer and desktop_app because they are python oriented 

## full Backend

- cd chemical_visualizer 
- python -m venv venv
- venv/Scripts/Activate
- docker compose up --build

## full frontend

- cd chemical_visualizer_frontend 
- npm install 
- npm run dev

## full desktop app

- cd desktop_app
- python -m venv venv
- venv/Scripts/Activate
- pip install -r requirements.txt
- python main.py
